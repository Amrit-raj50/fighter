# 🥊 Fighting Game (Canvas Based)

A 2D **Fighting Game** built using **HTML5 Canvas, CSS, and JavaScript**.  
Control your player, perform attacks, defeat the enemy, and manage health in real-time.

---

## 🎮 Gameplay

- Control a player character  
- Perform multiple attacks (Punch, Kick, Combo)  
- Enemy can move and attack  
- Health system for both player and enemy  
- Game ends when health is exhausted  

---

## 🚀 Features

- 🧍 Player movement (left/right + jump)  
- 🥊 Multiple attack animations  
- 🤖 Enemy controls (for testing)  
- ❤️ Health bar system  
- 🔄 Sprite-based animations  
- 🔊 Sound effects (kick & punch)  
- ⚡ Collision detection system  

---

## 🛠️ Tech Stack

- HTML5 Canvas  
- CSS3  
- JavaScript (Vanilla JS)

## 🎮 Controls

### 🧍 Player

W → Move Right  
S → Move Left  
Shift → Jump  
1 → Attack 1 (Punch)  
2 → Attack 2 (Combo Punch)  
3 → Attack 3 (Kick)  

---

### 🤖 Enemy (Manual Testing)

4 → Move Left  
6 → Move Right  
7 → Attack 1  
9 → Attack 2  

---

## ⚔️ Game Logic

- When player and enemy are close → attacks register  
- Each hit reduces opponent health  
- Sound plays on attack  
- When health reaches limit → character dies  

---

## 🧠 Concepts Used

- Canvas Rendering (`drawImage`)  
- Sprite Animation (frame-based)  
- Collision Detection  
- Keyboard Event Handling  
- Game Loop using `requestAnimationFrame()`
